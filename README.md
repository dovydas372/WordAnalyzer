# WordAnalyzer

Console aplikacija, kuri nuskaito tekstinį failą, suskaičiuoja skirtingų žodžių pasikartojimus ir išveda rezultatą mažėjančia tvarka pagal pasikartojimų skaičių.

## Features
- Skaito tekstą iš failo
- Normalizuoja žodžius (pvz. mažosios raidės, ignoruojama skyryba)
- Skaičiuoja žodžių pasikartojimus
- Išveda žodžius ir jų kiekius descending tvarka

## Requirements
- .NET: [užpildyk, pvz. .NET 8.0]
- IDE: Visual Studio [užpildyk versiją, pvz. 2022]

## How to run
### Visual Studio
1. Open solution `WordAnalyzer.sln`
2. Run project (Start / F5)
